# Calling

本项目是一个**本地最小化 LLM 网页对话框**（带工具 Agent），用于用你自己的 API 进行调用。

保留能力（按你的需求裁剪）：
- **单一主模型**：只做对话生成（可在 UI 里选 model 名称）。
- **20 轮上下文窗口**：只保留最近 20 轮（用户+助手）。
- **工具（Function Calling）**：
  - `web_search`：全网搜索（默认 DuckDuckGo）。
  - `read_file`：读取本项目目录下文件（用于让模型自行检索代码/文档）。
- **文件上传**：上传文件会复制到项目目录的 `data/uploads/`。
- **Embedding + RAG（开关）**：开启时，对上传文件做最基础的分块 embedding，并在**单个对话**内检索注入。
- **强制先联网（开关）**：开启时，本轮回答前模型必须先调用一次 `web_search`，否则后端会要求模型重试直到满足。

## 运行

### 1) 安装依赖

```bash
pip install -r requirements.txt
```

### 2) 配置环境变量

复制 `.env.example` 为 `.env`，填入你的 OpenAI 兼容接口配置（包括自建/代理都可以）。

### 3) 启动

```bash
python -m backend.main
```

然后打开 `http://127.0.0.1:8000/`。

## 说明

- **Web 搜索实现**：用 `duckduckgo-search`（无站点限制）。
- **文件读取权限**：只允许读取 `Calling/` 项目根目录内的相对路径（拒绝越权路径）。
- **RAG 存储**：本地 SQLite（`data/rag.sqlite3`），向量相似度用余弦相似度做最小实现（不引入向量数据库）。
